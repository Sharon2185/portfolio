## ✨ Features

- **3D Interactive Background** — Dynamic particle system and animated geometric shapes powered by Three.js
- **Smooth Inertial Scrolling** — Momentum-based scrolling experience using Inertia.js
- **Glassmorphism UI** — Modern frosted glass effect cards with subtle hover animations
- **Fully Responsive** — Seamless experience across desktop, tablet, and mobile devices
- **Interactive Elements** — Mouse-parallax cards, orbiting particles, and camera movement
- **Mobile Navigation** — Hamburger menu for optimal mobile experience
- **Performance Optimized** — Efficient rendering with Three.js and minimal repaints

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Three.js (r128)** | 3D graphics, particle systems, and immersive background animations |
| **Tailwind CSS v3** | Utility-first styling and responsive design |
| **Inertia.js** | Smooth scrolling with momentum and easing effects |
| **Vanilla JavaScript** | Core interactivity and animations |
| **HTML5/CSS3** | Structure and custom styling |
